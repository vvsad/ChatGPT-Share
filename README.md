# ChatGPT-Share

一个基于[cockroachai](https://github.com/cockroachai/cockroachai)开放能力实现的一个站点，实现ChatGPT的共享服务。

[演示站点](https://oai.xiamis.xyz) 

## 功能特点

- 官网UI1:1还原。
- 一键换车
- 无需科学上网，微信登录即可
- 对话隔离，他人无法查看你的对话内容，保护您的隐私
- 支持 ChatGPT 插件
- 支持 GPTs 商店
- 支持文件、照片、数据等分析
- 支持联网搜索对话
